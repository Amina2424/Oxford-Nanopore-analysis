# Анализ данных нанопорового секвенирования для выявления импринтированных регионов

Репозиторий содержит код для анализа данных нанопорового секвенирования с целью идентификации регионов геномного импринтинга.

## Структура репозитория

```
Imprintome/
├── methylation_modkit/         # Подсчет уровня метилирования
│   ├── main.nf/                # Основной пайплайн
│   └── nextflow.config/        # Модули пайплайна
├── monoallelic_regions/        # Папка с полученными результатами моноаллельных регионов
│   └── main                    # Описание содержимого
├── scripts/                    # Скрипты анализа
│   ├── bootstrap.py            # Скрипт для подсчета доверительного интервала
│   ├── candidate_search.py     # Скрипт для поиска моноаллельных регионов по заданным параметрам
│   ├── search_nearest_gene.py  # Скрипт для поиска близлежащих транскриптов
│   ├── meth_level.R            # Скрипты для визуализации промежуточных результатов
│   ├── meth_hist.R            
│   └── analysis_of_dmrs.R
├── vcf_phase/                  # Скрипт для анализа статистики фазирования по VCF-файлам
│   ├── main.nf                 # Основной пайплайн
│   └── nextflow.config         # Модули пайплайна
├── whatshap/                   # Поиск аллель-специфических регионов
│   ├── main.nf                 # Основной пайплайн
│   ├── nextflow.config         # Модули пайплайна
│   └── children.tsv            # Входной датасет для анализа
```

### Форматы файлов:
- `BAM (Binary Alignment Map)` - файл с выровненными прочтениями
- `BEDmethyl (Browser Extensible Data methylation format)` - файл, содержащий данные метилирования в каждом CpG-динуклеотиде по геному
- `VCF (Variant Call Format)` - файл с генетическими вариантами
- `PED` (Pedigree) - файл структурного описания родственных связей между образцами каждой семьи
- `*.ipynb' , '*.R' - скрипты для анализа


### Параметры анализа

### Критерии отбора кандидатных регионов:
- **Длина региона**: до 6000 п.н.
- **Минимальное количество CpG**: 5 сайтов
- **Уровень метилирования**: 43.5% - 67%
- **Минимальный score надежности**: 0.7

### Статистические параметры:
- Доверительный интервал: 99%
- Метод бутстрепа: 100000 ресемплирований
- Порог значимости: p-value < 0.001
